# cordova-plugin-lottie-splashscreen
Cordova plugin to show a bodymovin animations as the splashscreen with Airbnb's Lottie wrapper

### Supported platforms

* **iOS**

### Installation

`$ cordova plugin add cordova-plugin-lottie-splashscreen`
